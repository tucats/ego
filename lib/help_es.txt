.topic introduction

Ego es un lenguaje de script basado en el lenguaje Go. Cuenta 
con extensiones a Go, como tipos dinámicos y construcciones 
try/catch.

Utiliza el comando "help topics" para obtener una lista de 
todos  los temas de ayuda de nivel superior disponibles. 
Utiliza el comando  "help <tema>" para obtener ayuda sobre 
un tema específico.

Ego está escrito y mantenido por Tom Cole, tomcole@mac.com.
Sugerencias, ideas o críticas deben ingresarse en Github en
el proyecto Ego, https://github.com/tucats/ego. Utiliza la 
función "Issues" para describir tu solicitud o comentario.
