Settings

        
Settings are saved as browser cookies and persist across sessions.

        

            
            

                Remember login
                Save the API token as a browser cookie (expires after 24 hours). When enabled, you will not need to log in again after closing this tab.
            

        

        

            
            

                Dark mode
                Use a dark color scheme for all tabs. The Code tab always uses its own dark theme regardless of this setting.
            

        

        

            
            

                Use passkeys
                Allow passkey (biometric / hardware key) login and registration. Turn off to use passwords only, even if the server supports passkeys.
